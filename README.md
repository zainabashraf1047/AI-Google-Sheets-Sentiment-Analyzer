# AI Google Sheets Sentiment Analyzer

This project uses Google Apps Script + Gemini API to analyze text data directly inside Google Sheets.

## Features
- Sentiment analysis (Positive/Negative/Neutral)
- AI summarization (20 words max)
- Automatic row processing

## Tech Stack
- Google Apps Script
- Gemini API
- JavaScript

## Setup
1. Clone repo using clasp
2. Add GEMINI_KEY in Script Properties
3. Open Google Sheet
4. Run Analyze All
